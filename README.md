# ApexcifyTechnology_Task3


📊 Passenger Satisfaction Survey Analysis

📌 Project Overview

This project analyzes survey data to understand passenger satisfaction levels using Python.
The dataset contains responses from passengers about their travel experience.

The analysis focuses on the Satisfaction column, which indicates whether passengers were satisfied or dissatisfied with the service.

The results are visualized using a bar chart to clearly display how many passengers selected each satisfaction level.

🎯 Objective

The main objectives of this project are:

Load survey data about user satisfaction levels

Count how many people selected each satisfaction level

Display the results using a bar chart visualization


🛠 Tools & Technologies

This project uses the following tools and libraries:

Python

Pandas – data loading and analysis

Seaborn – data visualization

Matplotlib – chart customization

📂 Dataset

The dataset used in this project is the Airline Passenger Satisfaction Dataset.

Source: Kaggle

Key column used in the analysis:

Column	Description

Satisfaction	Indicates whether passengers were satisfied or dissatisfied


Possible values include:

Very Satisfied

Satisfied

Neutral

Dissatisfied

Very Dissatisfied
